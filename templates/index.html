SubSource Stremio Addon

This addon provides subtitles from SubSource, a multi-language subtitle provider.

Configuration

This addon requires a SubSource API key. You must set the SUBSOURCE_API_KEY environment variable before running the addon.

Installation

To install this addon, copy the following URL and paste it into the Stremio addon search bar:

{{ request.url_root }}manifest.json

Alternatively, you can click the button below to install the addon directly in Stremio.
